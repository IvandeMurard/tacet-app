Notes de conception · Itération 3

→ Carte : fond clair = #EDE8DF (crème chaud, Mapbox Light style). Fond sombre = #0C1524 (navy profond, Mapbox Dark style). Zones en halos doux via SVG feGaussianBlur stdDeviation="8" — pas de blocs opaques. Zone sélectionnée : pas de blur, bord net = précision visuelle.
→ Transparence popup : mode clair = bg-white/82 backdrop-blur-xl. Mode sombre = bg-white/7 backdrop-blur-24px + bordure rgba(255,255,255,0.14) légèrement plus définie. La carte navy contraste mieux avec le panneau semi-transparent.
→ dB bruts absents : sans contexte acoustique, « 66 dB » ne dit rien à Maria. Vue par défaut = score + label + note. Expert mode = Paramètres.
→ SearchBar : autofocus desktop uniquement. Mobile : pas d'autofocus — le clavier masquerait la carte.
